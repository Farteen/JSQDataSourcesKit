CollectionSupplementaryViewFactory

A CollectionSupplementaryViewFactory is a concrete CollectionSupplementaryViewFactoryType. This factory is responsible for producing and configuring supplementary views for a collection view for a specific data item.

The factory has the following type parameters:
<SupplementaryView: UICollectionReusableView, DataItem>

  • reuseIdentifier

    A unique identifier that describes the purpose of the supplementary views that the factory produces. The factory dequeues supplementary views from the collection view with this reuse identifier. Clients are responsible for registering a view for this identifier and a supplementary view kind with the collection view.
