CollectionViewFetchedResultsDelegateProvider

A CollectionViewFetchedResultsDelegateProvider is responsible for providing a delegate object for an instance of NSFetchedResultsController that manages data to display in a collection view.

The delegate provider has the following type parameters:
<DataItem>
Here, the DataItem type parameter acts as a phatom type. This type should correpsond to the type of objects that the NSFetchedResultsController fetches.
